### 第 7 期：ES5/ES6 的继承除了写法以外还有什么区别？

我的答案：
es6继承
```
class A extends B{}
2 A.__proto__ === B;  //继承属性
3 A.prototype.__proto__ == B.prototype;//继承方法
```
es5继承：
```
function Super() {}
function Sub() {}

Sub.prototype = new Super();
Sub.prototype.constructor = Sub;

var sub = new Sub();

Sub.__proto__ === Function.prototype;
```

思考：主要考虑es5与es6的继承方法不同
